# Presets

## Confirmed Default

Use this when the user asks for the current preferred style:

```bash
--size 800x960
--duration 7.5
--fps 30
--background black
--sticker-width 360
--sticker-x 395
--sticker-y 45
```

The confirmed style is black canvas, no blurred background fill, no color adjustment, original image centered, animated sticker in the top safe area, and random music segment.

Default local asset folders:

```text
~/Desktop/常用音乐
~/Desktop/常用贴纸
```

## Smaller Sticker

Use when the sticker starts to distract from the content:

```bash
--sticker-width 260
```

## More Visible Sticker

Use when GIF stickers have a lot of transparent padding:

```bash
--sticker-width 520
```
