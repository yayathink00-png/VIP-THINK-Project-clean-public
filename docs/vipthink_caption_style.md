# VIP THINK 社媒文案风格指纹

适用：Facebook / Instagram 图文、YouTube Shorts 标题与描述。

来源：用户提供的 5 条 HK 账号原始文案样本，包括等量代换、迷宫、火柴棒、密码推理。

## 核心判断

VIP THINK 原文风格不是“冷静教育科普”，也不是“硬广转化文”。它是：

- 儿童益智挑战感
- 暑假 / 每日练习 / 亲子陪玩语境
- 高 emoji 密度但有固定位置
- 轻粤语 + 繁体中文
- 前 80% 给题目价值和能力训练，最后 20% 才放互动与体验课链接

写文案时不要把营销完全拿掉，也不要让整篇都像广告。营销只能放在尾部 1-2 行。

## HK 文风指纹

### 语言

- 繁体中文为主。
- 可混入轻粤语：`搵`、`睇`、`唔好`、`即刻`、`快啲`、`同小朋友一齊`、`細路`、`千祈唔好`。
- 保留少量大陆运营常用词也可以，例如：`核心邏輯`、`抽象推理能力`、`底層數理思維`、`系統提升`。
- 不要写成过度书面教育论文。

### 语气

- 活泼、热闹、挑战式。
- 像“每日益智挑战 + 家长陪练提醒”。
- 可以有轻度营销，但正文主体要先讲玩法、题型、能力训练。

### 常见词库

- 主题词：`暑假`、`假期`、`每日`、`腦力`、`動腦挑戰`、`思維訓練`、`數理小探員`、`燒腦題`、`闖關`、`解謎`、`小貼士`。
- 能力词：`邏輯推理`、`觀察力`、`專注力`、`空間認知`、`抽象推理`、`多角度解題`、`數學思維`。
- 情绪词：`好玩又益智`、`唔枯燥`、`滿足感爆棚`、`輕鬆拆解`、`一齊動腦`。
- 互动词：`留言分享`、`評論區留言`、`寫出答案`、`分享你嘅解題步驟`。

## 结构模板

```text
【主题 emoji + 题型/挑战名 + 结尾 emoji】
副标题/一句玩法说明

场景 + 难点：
说明这张图是什么挑战，孩子哪里容易卡住。

价值/能力：
一边玩/做题，一边训练 2-4 个能力。

解题/闯关小贴士：
给 1-3 句具体方法，避免空泛。

互动：
邀请家长和孩子留言答案 / 分享步骤。

轻营销：
想体验更多类似题 / 系统提升思维，可以点链接。

固定账号块 + hashtag
```

## Emoji 位置与数量

原样本 emoji 密度较高。不要压得太素。

建议：

- 短文：12-18 个 emoji / 符号。
- 长文：18-28 个 emoji / 符号。
- `✔` 列表通常 3 条，每条 1 个 `✔`，句尾可再有 1 个功能 emoji。

固定位置：

- 标题：2-4 个 emoji，常见首尾包裹。
  - `【🍧暑假等量代換重難點必刷題🔥🔥🔥】`
  - `【🐭幫小老鼠搵芝士🧀動腦挑戰！】`
  - `🔥【暑假腦力充電計劃🍧】`
  - `【🔒密碼推理大闖關🔑】`
- 第一段：1-3 个 emoji，用于题型、难点、情绪。
  - `⌛暑期思維養成黃金階段...😵‍💫`
  - `🐢逐個試數浪費時間...❌`
- 能力列表：用 `✔` 开头，句尾放能力相关 emoji。
  - `✔ 靜心細察📝，鍛煉長時間專注力✨`
  - `✔ 分辨方向🎯，強化空間認知能力🧩`
- 小贴士：常用 `💡`、`📌`、`🗺️`。
- 互动：常用 `💬`、`🗨️`、`👇`。
- 轻营销：常用 `👉`、`👩🏻‍🏫`、`🎁`，但只放尾部。

避免：

- 不要把每一行都写成独立广告口号。
- 不要一开头就预约体验课。
- 不要把正文写得太冷静、太像教材解析。
- 不要把 HK 文案写成纯普通话繁体。

## Global 改写规则

Global 使用普通话 / 简体中文，但保留同样的结构和热闹感。

- `搵` → `找`
- `睇` → `看`
- `唔好` → `不要`
- `即刻` → `马上 / 立刻`
- `快啲` → `快来`
- `細路 / 小朋友` → `孩子 / 小朋友`
- `同小朋友一齊` → `和孩子一起`

不要翻译成英文，不要写成过度正式的课程广告。
