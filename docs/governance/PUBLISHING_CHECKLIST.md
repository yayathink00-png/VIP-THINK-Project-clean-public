# Publishing Checklist

Before release:

- [ ] Maintainer has confirmed the intended public scope.
- [ ] Only whitelist files are included.
- [ ] No raw assets, exports, archives, generated batches, logs, caches, or local databases are included.
- [ ] No unreviewed images, video, audio, PDF, HTML, CSV, spreadsheet, or ZIP files are included.
- [ ] No real tokens, cookies, API keys, app tokens, table IDs, spreadsheet tokens, webhooks, or private URLs are included.
- [ ] Sample data is synthetic or fully anonymized.
- [ ] README states public/private boundaries.
- [ ] Contribution boundary note is present.
- [ ] Asset governance rules are present.
- [ ] Security checklist has passed.
- [ ] PR description includes responsibility boundary and excluded content.
- [ ] No push, force push, visibility change, remote deletion, or history rewrite occurs without explicit authorization.
