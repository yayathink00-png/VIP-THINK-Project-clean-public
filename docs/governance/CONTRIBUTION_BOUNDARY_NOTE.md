# Contribution Boundary Note

This repository is maintained for public-safe source code, documentation, templates, and synthetic examples.

Contributors must not submit:
- secrets, tokens, cookies, passwords, private keys, or webhooks
- real user, customer, student, parent, supplier, order, contract, revenue, cost, or account data
- raw captures, exports, logs, caches, generated batches, or local databases
- private prompts, source-media breakdowns, or AI intermediate artifacts
- unreviewed images, videos, audio files, PDFs, HTML exports, CSV exports, or archives

Public contribution review covers only the proposed change. It does not certify historical commits, forks, clones, third-party redistribution, or prior public exposure.
