## 命名规则

图片文件名格式：

```text
序号.语言.jpg
```

- 前面的数字：内容序号 / 当周第几条内容。
- `.1`：粤语 / 繁体，用于港澳账号。
- `.2`：普通话 / 简体，用于 Global / 非港澳账号。

例如：

```text
1.1.jpg = 第 1 条内容，粤语版
1.2.jpg = 第 1 条内容，普通话版
```
