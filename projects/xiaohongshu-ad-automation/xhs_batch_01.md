孩子不是笨，是没学会拆题

## XHS-002

Type: Learning method

Title options:
- 数学启蒙别急着刷题，先走这3步
- CPA教学法，适合数学刚启蒙的孩子
- 孩子数学不开窍，可以先这样练

Cover text:
数学启蒙别急着刷题

Carousel:
1. 数学启蒙不是一开始就刷题。
2. C 是 concrete：先用实物理解。
3. P 是 pictorial：再用图形表达。
4. A 是 abstract：最后进入数字和公式。
5. 孩子跳过前两步，容易只会背不会用。
6. 家长可以用积木、水果、图形先讲清楚。
7. 先理解，再计算。

Body:
很多孩子数学学得吃力，是因为太早进入抽象符号。比如加减法，孩子还没理解“合起来”“拿走”的意思，就开始背算式，很容易变成机械记忆。可以试试 CPA 顺序：先用实物摆一摆，再画图表示，最后才写数字和算式。这样孩子不是在背答案，而是在理解数量关系。数学启蒙阶段，慢一点反而更稳。

## XHS-003

Type: Mistake correction

Title options:
- 只盯答案，孩子会越来越怕数学
- 陪作业时，别急着问“答案是多少”
- 数学真正要练的不是答案，是过程

Cover text:
别只盯答案

Carousel:
1. 很多家长陪作业，第一反应是看答案。
2. 但孩子最需要被看见的是思考过程。
3. 答案错了，可能只是一步理解错。
4. 答案对了，也可能是猜对。
5. 让孩子说出“为什么这样做”。
6. 过程清楚，成绩才有机会稳定。
7. 今天开始，多问一句：你怎么想的？

Body:
陪孩子做数学时，很多家长会忍不住直接看答案对不对。但对孩子来说，真正需要训练的是思考过程。答案错了，不代表完全不会；答案对了，也不代表真的理解。建议家长把“答案是多少”换成“你是怎么想的”。如果孩子能说清楚条件、步骤和理由，就算最后算错，也很容易修正。数学思维不是靠催出来的，是在一次次表达思路中建立起来的。

Title options:

- 会讲思路，比做对一道题更重要

Cover text:
会讲思路更重要

Carousel:
1. 有些孩子做题喜欢猜。
2. 猜对了很开心，猜错了就崩溃。
3. 真正要练的是表达思路。
4. 先让孩子说条件，再说方法。
5. 不急着纠错，先听完整。
6. 孩子能解释，才是真的开始理解。
7. 家长要奖励思考，不只奖励正确。

Body:
我们观察到，很多孩子不是完全不会数学，而是习惯“猜答案”。一旦猜错，就觉得自己不行。更好的训练方式，是让孩子先说思路：这题有哪些条件？你打算先做什么？为什么这样做？刚开始孩子可能说得慢、说不完整，但这正是思维在形成。家长不要急着打断，可以先听完，再帮孩子补充。会讲思路的孩子，后面遇到新题会更有底气。

Title options:
- 孩子一遇难题就放弃，怎么办？
- 孩子不是不努力，是缺少解题抓手
- 难题面前，先别急着讲答案

Cover text:
一遇难题就放弃？

Carousel:
1. 孩子遇到难题就放弃，不一定是懒。
2. 可能是他不知道第一步怎么做。
3. 家长可以先降低启动难度。
4. 问：这题你看懂了哪一句？
5. 再问：有没有和以前见过的题相似？
6. 让孩子先完成一小步。
7. 启动了，才有机会继续思考。

Body:
孩子一遇到难题就说“我不会”，很多时候不是不努力，而是没有解题抓手。家长这时不要马上讲完整答案，可以先帮孩子找到第一小步。比如问：这题你看懂了哪一句？它和以前哪类题有点像？你能先画个图吗？当孩子能先动起来，焦虑会下降，思考才会继续。训练孩子面对难题，重点不是立刻做对，而是学会开始。

Title options:
- 家长这样问，孩子更愿意思考
- 陪孩子学习，少讲答案多问问题
- 3个问题，帮孩子打开思路

Cover text:
这样问，孩子更愿意思考

Carousel:
1. 好问题比直接讲答案更有用。
2. 问题1：你看到了哪些条件？
3. 问题2：你准备先做哪一步？
4. 问题3：还有没有别的方法？
5. 不急着评价，先让孩子说完。
6. 孩子表达时，思路会更清楚。
7. 家长少讲一点，孩子多想一点。

Body:
陪孩子学习时，家长讲得越多，孩子可能越依赖。可以把讲答案换成提问题。先问“你看到了哪些条件”，帮助孩子读题；再问“你准备先做哪一步”，帮助孩子启动；最后问“还有没有别的方法”，训练灵活思考。刚开始孩子可能回答得不完整，但只要愿意说，就已经在练思维。家长的目标不是替孩子完成题目，而是帮孩子建立方法。

编程启蒙先练逻辑

在家练逻辑的3个小游戏
