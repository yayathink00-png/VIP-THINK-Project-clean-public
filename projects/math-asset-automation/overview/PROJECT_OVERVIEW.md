# Project Overview

## Goal

Build an automation workflow that turns math-thinking content requirements into managed image-production tasks, generated assets, upload status, and structured feedback.

## User Value

The project helps the team manage image material production at scale. It reduces scattered communication, makes asset status visible, and creates a path for consistent production and review.

## Scope

Included in the working project:
- task-based material requirement management
- status inspection
- output attachment upload workflow
- status update workflow
- team collaboration instructions
- image generation handoff process

Excluded from this clean repo:
- generated images
- packaged image ZIPs
- production platform URLs
- app/table/spreadsheet identifiers
- real attachment records
- private handoff notes

## Leadership Message

This work shows a move from ad hoc image creation to a structured asset-production workflow, with clearer task ownership and status tracking.
