# Math Asset Automation

## One-Line Summary

将数学思维素材需求整理为图片生成、状态跟踪、附件上传和结果回填的自动化流程。

## Business Problem

素材生产需要把需求、生成、审核、上传和状态更新统一管理。没有流程化工具时，容易出现素材版本混乱、回填遗漏和协作信息分散。

## Delivered Capability

- 将素材需求按任务行管理
- 支持读取素材方向和状态
- 支持上传产出附件
- 支持更新素材任务状态
- 建立团队协作说明和后续自动化路径
